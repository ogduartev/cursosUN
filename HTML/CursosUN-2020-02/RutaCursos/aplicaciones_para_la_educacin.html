Aplicaciones para la educación

Una clasificación

  • Aplicaciones para la instrucción: son aquellas usadas para exponer contenidos de cualquier tipo, que el estudiante puede explorar de manera autónoma o guiada (por el docente). Ejemplos son las presentaciones de clase (en diferentes formatos), los videos educativos o los mapas conceptuales.
  • Aplicaciones para la práctica o la producción: en este grupo se clasifican aquellas cuyo propósito es afianzar los aprendizajes mediante la práctica o la producción de objetos digitales. Este tipo de aplicaciones favorecen el desarrollo de habilidades de orden superior como el análisis, la evaluación y la creación. Entre las aplicaciones  para la práctica se encuentran las simulaciones o los laboratorios virtuales y para la producción se tienen plataformas y software para la creación de infografías, animaciones, comics o muros digitales, entre otras.
  • Aplicaciones para la colaboración o la comunicación: son aquellas diseñadas para fomentar el trabajo colaborativo y la interacción dinámica en entornos virtuales. 
  • Aplicaciones para la evaluación: son aquellas que pueden ser utilizadas en el proceso de evaluación de los aprendizajes.
